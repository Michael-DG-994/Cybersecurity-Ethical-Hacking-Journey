
# Gemini Prompt-Injection Guard
Esercizio dove tramite pattern REGEX e normalizzazione di input lo script identifica tentativi di Prompt-Injection e risponde in maniera adeguata per evitare leak di dati sensibili in un "database" simulato.

### DISCLAIMER
Necessario includere la propia KEY API di gemini nell'enviroment, modo in cui è stato fatto sulla mia macchina è estremamente non convenzionale ma non so ancora programmare e questo esercizio è stato svolto principalmente per imparare i principi base del promp-injection e di REGEX
